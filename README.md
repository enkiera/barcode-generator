# Modern Barcode Generator

A lightweight, premium-styled barcode generator web application. Built with HTML5, Vanilla CSS, and `bwip-js`.

## Features
- **Modern UI**: Dark mode, glassmorphism aesthetics.
- **Many Barcode Types**: Supports Code 128, EAN-13, UPC-A, QR Code, Data Matrix, and more.
- **Customization**:
    - Adjustable height (for 1D barcodes).
    - Adjustable scale.
    - Option to include/hide human-readable text.
- **Real-time**: Generates barcodes as you type.
- **Download**: Export generic PNG images easily.

## Usage
Simply open `index.html` in your web browser. No installation or server required.

## Technologies
- **HTML5**: Semantic structure.
- **Vanilla CSS**: Responsive, flexbox/grid, custom properties.
- **bwip-js**: Robust barcode generation library (loaded via CDN).
- **Vanilla JS**: No frameworks, just speed.
